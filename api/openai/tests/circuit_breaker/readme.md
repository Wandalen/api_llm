# circuit_breaker

Circuit breaker pattern tests organized by test domain.

## Purpose

Tests for circuit breaker reliability feature, organized by functional area rather than test methodology.

## Contents

- `config_tests.rs` - Configuration validation tests
- `recovery_tests.rs` - Recovery behavior tests
- `state_transition_tests.rs` - State machine transition tests
- `mod.rs` - Module organization

## Test Organization

Tests are organized by functional domain (what is being tested) rather than by test type, following the project's domain-based test organization standards.
